About

Welcome to the Utility App! This web application offers a variety of tools to make your life easier:

  • Text to QR Code - Convert any text into a QR code instantly. 📱
  • Base64 Encode/Decode - Encode or decode text using Base64. 🔐
  • Hash Generator - Generate hashes using various algorithms like MD5, SHA-1, SHA-256, and SHA-512. 🔒
  • SSH Key Generator - Generate SSH key pairs for secure communication. 🔑
  • Markdown to PDF - Convert your Markdown text to a PDF document. 📄
  • JSON/XML Converter - Convert JSON data to XML format and vice versa. 🔄
  • JWT Decoder - Decode JSON Web Tokens (JWT) to view their contents. 🛡️
  • JSON Structure Viewer - Visualize the structure of your JSON data. 🌐
  • Color Picker - Pick colors and convert between RGB and HEX formats. 🎨

Explore these features and enhance your productivity with our Utility App! 🚀
